# CASE — Design Tokens

## Colors
| Token | Hex | Usage |
|---|---|---|
| `--bg-primary` | `#0F172A` | Page background |
| `--bg-card` | `#1E293B` | Cards, panels |
| `--accent-blue` | `#3B82F6` | CTA, icons, glow |
| `--accent-purple` | `#8B5CF6` | Gradient complement |
| `--text-primary` | `#F1F5F9` | Headlines, body |
| `--text-muted` | `#94A3B8` | Supporting text |
| `--border-subtle` | `rgba(255,255,255,0.08)` | Card borders |
| `--glow-blue` | `rgba(59,130,246,0.3)` | Glow shadows |
